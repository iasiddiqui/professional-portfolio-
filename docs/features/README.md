# Features

## Core Features

| Feature | Description | Priority |
| ------- | ----------- | -------- |
| **Public Portfolio** | SEO-friendly public profile with projects showcase | P0 |
| **Project Management** | CRUD for portfolio projects with images and tags | P0 |
| **Authentication** | Register, login, JWT session management | P0 |
| **Contact Form** | Public contact submission with validation | P1 |
| **Dashboard** | Admin UI for content management | P0 |
| **Theme Customization** | Portfolio appearance settings | P2 |
| **Analytics** | Page view tracking | P3 |
